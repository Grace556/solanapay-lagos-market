# SolanaPay Lagos Market
Solana Foundation Grant 2026 Application

A mobile payment solution for Lagos market traders using Solana USDC/SOL QR payments.

## Features
- QR code payments with zero fees
- SMS receipts for offline confirmation  
- NGN to USDC conversion guide for traders

## Milestones
- MVP with QR payments - 3 weeks
- Pilot with 30 traders - 4 weeks  
- Public launch with docs - 3 weeks

Built for financial inclusion in Nigeria.
